To-Do List application built using Kotlin and Jetpack Compose.

1)Features
Add tasks instantly
Delete tasks with one tap
Clean dark UI with purple accent theme
Rounded modern card layout
State driven UI using Compose

2)Tech Stack
Kotlin
Jetpack Compose
Material 3
State management using remember & mutableStateOf

3)UI Design
Minimal dark theme
Purple accent highlights
Rounded 24dp cards
Bold typography hierarchy

4)How to Run
Download the ZIP file
Extract it
Open in Android Studio
Run on emulator or physical device
